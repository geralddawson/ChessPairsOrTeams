# ChessPairsOrTeams

A macOS app built with Swift and Xcode to run chess tournaments using pairs or team of 4 formats.

## Features

- Manage teams and players
- Generate pairings - Swiss style draw for individual or teams
- Odd number of players or teams handled with bye rounds 
- Track tournament progress
- Player substitutions handled
- Some colour balancing if required
- Tie breaks - Buchhloz 
- Save and export results to file and create spreadsheets thereof 
- Designed for school and club events

## Requirements

- macOS
- Xcode (latest recommended)

## How to Run

1. Download or clone this repository
2. Open `ChessPairsOrTeams.xcodeproj` in Xcode
3. Press ▶️ Run

## Notes

This app is part of an ongoing project to develop tools for running chess tournaments, particularly in school environments.
